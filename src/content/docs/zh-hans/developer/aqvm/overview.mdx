## 简介
AQ 的虚拟机是为运行和调试 AQ 字节码而设计的高性能虚拟环境。它提供了一个抽象的执行层，可以跨多个平台无缝运行 AQ 字节码。该虚拟机支持内存管理、垃圾回收、多线程并发等功能，同时优化了指令执行速度和资源利用效率。通过该虚拟机，开发者能够在不同的硬件和操作系统上运行一致的 AQ 字节码，为跨平台开发提供了强大的支持。<br/>

## 开发进度
AQ 虚拟机作为 AQ 的核心组件，在 AQ 开发计划经历多次修改后成为了目前开发优先级最高的组件。<br/>

但由于开发时间较慢和计划的变动，此后又经历了多次改版。<br/>

目前决定放弃原有的代码，重新开始原型设计，在原型设计的基础上不断完善，直到开发完成。<br/>

您可以在 Github 查看代码或加入开发工作。<br/>
